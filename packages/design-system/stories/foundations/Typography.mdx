# Typography

Typography is the primary brand signal. The pairing is deliberate:
**Söhne** (or General Sans) for authority and readability;
**Berkeley Mono** (or Commit Mono) for technical credibility and wayfinding.

Never Inter. Never Geist. Never Google Fonts CDN — self-host only.

## Font pairing

| Role | Preferred (paid) | Fallback (free) | Never |
|------|-----------------|-----------------|-------|
| Sans (primary) | **Söhne** — Klim Type | **General Sans** — Fontshare | Inter, Geist |
| Mono (accent) | **Berkeley Mono** — U.S. Graphics | **Commit Mono** | JetBrains Mono |

Fonts are self-hosted as `woff2` from `/public/fonts/`. See `src/styles/base.css` for `@font-face` declarations.

## Rules

- **Display & heading**: weight 500 (medium). Never 700+. Avoid bro-tech heaviness.
- **Body**: weight 400. Use 500 for emphasis only. Italic is meaningful — for *terms of art*, ship names, meta-commentary. Never for punchlines.
- **Mono**: weight 400. For labeling and technical artifacts only. Never decorative running text.
- Letter-spacing: `-0.01em` on display sizes, `0` on body, `+0.04em` on uppercase mono labels.
- Default text color is `color.ink.primary` (`#1F1B16`). Never set text to `#000`.
