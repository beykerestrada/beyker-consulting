# Spacing, Radii, Shadows

## Spacing scale (4-based)

Default vertical rhythm between content blocks is `space.7` (48px). Layout breathes.

## Radii

No radii larger than 12px on surfaces. The look is paper, not balloons. `pill` is reserved for tags and badges only.

| Token | Value | Use |
|-------|-------|-----|
| `radius.none` | 0px | Sharp corners |
| `radius.sm` | 4px | Subtle — inputs, small elements |
| `radius.md` | 8px | Default — cards, buttons |
| `radius.lg` | 12px | Maximum surface rounding |
| `radius.pill` | 999px | Tags, badges **only** |

## Shadows

Minimal. Borders do the work, not shadows. A card looks like a card because of its edge, not its halo.

| Token | Value | Use |
|-------|-------|-----|
| `shadow.none` | none | Default for cards |
| `shadow.sm` | `0 1px 2px rgba(31,27,22,0.06)` | Subtle float |
| `shadow.md` | `0 2px 8px rgba(31,27,22,0.08)` | Dropdowns, toasts |

No `shadow.lg`. No glow effects. No colored shadows.

## Borders

Default border: `1px solid color.border.default` (`#C7C0AE`).

Hairline emphasis is preferred over shadow. Use borders to define edges, whitespace to create hierarchy.
